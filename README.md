## Getting Started

### Prerequisites

* npm
  ```sh
  npm install -g yarn@latest
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sevenhist/pizzaiolo.git
   ```
2. Install yarn packages
   ```sh
   yarn install
   ```
3. start the server
    ```sh
   yarn start
   ```
